# Connect-4
Local Player vs. Player game of Connect 4. This program allows two players to play Connect 4 against each other wherein players take turns placing an 'X' (player 1), or an 'O' (player 2) into a vertical grid. The goal is to be the first to connect four 'X's' or 'O's' in a row, either horizontally, vertically, or diagonally.
## How to use
Compile and run the C++ code on any C++ compiler. Online GDB is a recommended online C++ compiler. When prompted, each player will take turns selecting a column between [0, 6] by pressing 0-6 to place their 'X' or 'O'. When a victory condition is met, the game ends, and the winning player is congratulated.
